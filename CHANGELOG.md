# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-10

### Added
- Initial release of JobPlatform
- User authentication with Supabase
- Job posting and browsing functionality
- Job application system
- User profiles with company information
- CV upload and management
- Job alerts and notifications
- Bookmark/save jobs feature
- Real-time application tracking
- Easy Apply modal for quick applications
- Advanced job filtering and search
- Responsive design with NativeWind
- Dark mode support
- Analytics integration with Amplitude
- Error tracking with Sentry
- TypeScript support
- Jest testing setup
- Detox E2E testing framework

### Features
- **Authentication**: Login and registration with email
- **Job Management**: Post, edit, and delete job listings
- **Applications**: Apply to jobs with CV and cover letter
- **Profiles**: User and company profile management
- **Search & Filter**: Advanced search with multiple filters
- **Notifications**: Push notifications for job alerts
- **Bookmarks**: Save jobs for later viewing
- **Analytics**: User behavior tracking and insights

### Technical
- React Native with Expo SDK 54
- TypeScript for type safety
- Supabase for backend and authentication
- NativeWind for styling
- React Query for data fetching
- Expo Router for navigation
- Jest for unit testing
- Detox for E2E testing
